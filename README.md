# C Systems Programming

## Purpose
Hands-on C programs focused on system-level programming concepts.

This repository covers:
- file descriptors and I/O
- process creation and management
- threads and synchronization
- socket programming
- basic networking concepts

## Structure
- src/   → C source code (organized by topic)
- notes/ → explanations and theory notes
- labs/  → practice tasks and mini projects

## Topics
- Basics (memory, pointers)
- Files (open/read/write)
- Processes (fork/exec/wait)
- Threads (pthreads)
- Sockets (TCP/UDP client-server)
